# 기술 스택

| 항목 | 선택 |
|------|------|
| Framework | Next.js 15 (App Router) + TypeScript |
| UI | shadcn/ui + Tailwind CSS |
| DnD | dnd-kit |
| 상태관리 | Zustand (Phase 1 로컬) → Firebase (Phase 2) |
| 실시간 | Firebase Firestore |
| 인증 | 로컬 Mock (Phase 1) → Google OAuth (Phase 2 마지막) |
| 테스트 | Jest + React Testing Library |
| 반응형 | 필수 (Tailwind 기준) |
| i18n | next-intl v3 (ko/en, cookie-based locale) |
